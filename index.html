Project On Console

Visual Studio Code is an IDE (Like Brackets) used to do coding without the help of the internet. Which means you can use Visual Studio Code anytime, anywhere. To download the IDE, simply search up 'Visual Studio Code' and click on the first option. Then you must click download and then click on which device you are using, once done simply open the app from your finder and you are done!

The console screen is for developers to analyse and store values by displaying them in the console>
